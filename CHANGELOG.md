# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.1] - 2026-03-13

- Do not use Uint8Array.prototype.toHex, which is only supported by recent browsers
- Update translations
- Update documentation screenshots

## [0.2.0] - 2026-03-12

- Make component label optional
- Fix a CKEditor-related bug when using multiple HTML components
- Clarify what 'name' is for a component
- Show help text by default and add per-component option to hide it
- Make the select's "empty option" configurable
- Fix client side admin form validation and validate component names
- Use 'email' type for UserEmail form component
- Add form component type Email
- Add "(required)" text next to required fields for accessibility
- Show form errors without using the messenger
- In submission search form, show only users that can return results
- Add ability to search by submitter email

## [0.1.1] - 2026-02-17

- Add french translation
- Add documentation

## [0.1.0] - 2026-02-11

Initial release

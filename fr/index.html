Documentation de Formularium

Formularium est un module pour Omeka S. Il permet aux administrateurs de construire des formulaires pour les sites publics.

Les soumissions de formulaires sont enregistrées et sont consultables depuis l’interface d’administration.

Soumettre un formulaire peut aussi déclencher plusieurs « actions ». Il n’y a qu’un seul type d’action pour le moment: envoyer un e-mail.

Objectifs

L’objectif principal de ce module est d’être une alternative viable aux formulaires spécifiques de type « contactez-nous ».

Il est aussi extensible afin que d’autres modules puissent ajouter leurs propres composants de formulaire et leurs propres actions.

Exigences

  • Omeka S >= 4.1.0

Démarrage rapide

  1. Ajouter le module à Omeka S

  2. Dans l’interface d’administration, cliquez sur « Formularium » dans le menu de navigation

  3. Créez un nouveau formulaire, donnez lui un nom et ajoutez des « composants » (éléments de formulaire)

  4. Éventuellement, ajoutez des actions

  5. Créez une nouvelle page pour votre site, et ajoutez le bloc « Formularium » à cette page. Dans les paramètres du bloc, sélectionnez le formulaire que vous avez créé.

Fonctionnalités

  • Créez une infinité de formulaires avec une infinité de composants et d’actions

  • Différents types de composants: texte, zone de texte, liste déroulante, case à cocher, antispam (recaptcha), chargement de fichier, … (d’autres modules peuvent ajouter les leurs)

  • Un type d’action: envoyer un e-mail (d’autres modules peuvent ajouter les leurs)

  • Les soumissions de formulaire sont entièrement sauvegardées (y compris les fichiers chargés) pour pouvoir être consultables ultérieurement. Supprimer une soumission de formulaire supprime également les fichiers attachés.

  • Une marque peut être posée pour marquer une soumission de formulaire comme étant « traitée » (quoi que ça signifie pour vous)

  • Les soumissions de formulaire peuvent être filtrées par formulaire, site, utilisateur (si un utilisateur authentifié a fait une soumission), et par status traité

  • Les soumissions de formulaires peuvent être supprimées et modifiées par lot (la modification par lot permet uniquement de modifier le statut traité)

Comparaison avec les modules similaires

Contact Us

  • Contact Us est plus facile à mettre en place. Formularium est plus personnalisable

  • Contact Us a deux solutions antispam: simple et recaptcha. Formularium a seulement recaptcha
